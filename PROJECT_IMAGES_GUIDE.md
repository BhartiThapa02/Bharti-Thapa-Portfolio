# Project Images Guide

## How to Add Project Images

To make your portfolio even more impressive, add real screenshots of your projects!

### Step 1: Create Projects Folder
Create a folder named `projects` in your portfolio directory:
```
portfolio/
  ├── projects/
  │   ├── job-portal.jpg
  │   ├── lms.jpg
  │   ├── attendance-system.jpg
  │   ├── cursor-oauth.jpg
  │   ├── jaarvis.jpg
  │   ├── invoice-generator.jpg
  │   ├── weather-app.jpg
  │   ├── qr-generator.jpg
  │   └── calculator.jpg
  └── ...
```

### Step 2: Take Screenshots
For each project, take high-quality screenshots:
- **Recommended size**: 1200x800px or 16:9 aspect ratio
- **Format**: JPG or PNG
- **Quality**: High resolution, clear and professional
- **Content**: Show the main features and best UI elements

### Step 3: Name Your Images
Name your images exactly as shown below:

| Project | Image Filename |
|---------|---------------|
| Job Portal | `job-portal.jpg` |
| Learning Management System | `lms.jpg` |
| Student Attendance System | `attendance-system.jpg` |
| CursorOAuth | `cursor-oauth.jpg` |
| Jaarvis Voice Assistant | `jaarvis.jpg` |
| Hotel Invoice Generator | `invoice-generator.jpg` |
| Weather App | `weather-app.jpg` |
| QR Code Generator | `qr-generator.jpg` |
| Modern Calculator | `calculator.jpg` |

### Step 4: Tips for Great Screenshots

1. **Show the Best Features**: Capture the most impressive parts of your project
2. **Clean UI**: Make sure the interface looks polished
3. **Good Lighting**: Use screenshots with good contrast
4. **Multiple Views**: Consider showing different pages/features
5. **Remove Personal Data**: Blur or remove any sensitive information

### Step 5: Image Optimization

Before adding images, optimize them:
- Compress images to reduce file size (use tools like TinyPNG)
- Keep file sizes under 500KB for fast loading
- Maintain good quality while reducing size

### Alternative: Using Placeholder Images

If you don't have screenshots yet, the portfolio will automatically show icon placeholders. You can add images later anytime!

### Tools for Screenshots

- **Windows**: Snipping Tool, Snip & Sketch, or ShareX
- **Mac**: Cmd + Shift + 4
- **Browser Extensions**: Full Page Screen Capture, Awesome Screenshot
- **Online Tools**: Screenshot.guru, Page2Images

---

**Note**: The portfolio is designed to gracefully handle missing images - it will show beautiful icon placeholders until you add your screenshots!
