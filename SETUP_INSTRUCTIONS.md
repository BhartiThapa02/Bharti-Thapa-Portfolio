# Quick Setup Instructions

## üöÄ Get Your Portfolio Live in 3 Steps

### Step 1: Add Your Avatar Image
1. Save your 3D character image as `avatar.png` in the portfolio folder
2. The image will automatically appear in the hero and about sections

### Step 2: Add Project Screenshots (Optional but Recommended)
1. Create a `projects` folder (already created for you!)
2. Add screenshots of your projects with these exact names:
   - `job-portal.jpg` - Your Job Portal project
   - `lms.jpg` - Your Learning Management System
   - `attendance-system.jpg` - Student Attendance System
   - `cursor-oauth.jpg` - CursorOAuth project
   - `jaarvis.jpg` - Jaarvis Voice Assistant
   - `invoice-generator.jpg` - Hotel Invoice Generator
   - `weather-app.jpg` - Weather App
   - `qr-generator.jpg` - QR Code Generator
   - `calculator.jpg` - Modern Calculator

**Tip**: Take high-quality screenshots (1200x800px recommended) showing the best features of each project!

### Step 3: Customize Your Information
1. Open `index.html` in a text editor
2. Update your email address in the contact section
3. Add your LinkedIn/Twitter URLs (if you have them)
4. Update project links with your actual GitHub URLs and live demo links

### Step 4: View Your Portfolio
- **Option 1**: Simply double-click `index.html` to open in your browser
- **Option 2**: Run a local server:
  ```bash
  python -m http.server 8000
  ```
  Then visit: `http://localhost:8000`
